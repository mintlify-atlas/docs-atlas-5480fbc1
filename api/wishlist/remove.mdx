---
title: "Remove from wishlist"
description: "Delete a wishlist entry by its entry ID."
api: "DELETE https://api.cafeacacias.com/api/wishlists/:id"
---

<Note>
  The `:id` path parameter is the **wishlist entry ID**, not the product ID. To remove a product by its ID, first query `GET /api/wishlists` filtered by `filters[product][id][$eq]={productId}` and `filters[user][id][$eq]={userId}` to retrieve the wishlist entry ID, then delete by that entry ID.
</Note>

<Warning>
  This action is irreversible. Deleting a wishlist entry cannot be undone.
</Warning>

## Authentication

<ParamField header="Authorization" type="string" required>
  Bearer token. Format: `Bearer <jwt_token>`
</ParamField>

## Path parameters

<ParamField path="id" type="number" required>
  Numeric ID of the wishlist entry to delete.
</ParamField>

## Response

Returns `200 OK` with the deleted wishlist entry object as it existed before deletion.

<ResponseField name="data" type="object">
  The deleted wishlist entry.

  <Expandable title="wishlist entry object">
    <ResponseField name="id" type="number">Numeric ID of the deleted wishlist entry.</ResponseField>
    <ResponseField name="documentId" type="string">Strapi v5 document ID.</ResponseField>
    <ResponseField name="createdAt" type="string">ISO 8601 creation timestamp.</ResponseField>
    <ResponseField name="updatedAt" type="string">ISO 8601 last-updated timestamp.</ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="meta" type="object">
  Empty object for single-resource responses.
</ResponseField>
