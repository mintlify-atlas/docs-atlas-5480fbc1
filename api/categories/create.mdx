---
title: "Create a category"
description: "Create a new product category. Requires admin-level authentication."
---

Creates a new category and publishes it immediately. The `slug` field is automatically generated from the `title` — you do not need to provide it.

<Warning>
  This endpoint requires an admin or content-manager role. Requests authenticated with a standard customer token will receive a `403 Forbidden` response.
</Warning>

<Note>
  The `slug` is auto-generated by Strapi from the `title` field using the UID field type. If you update the title later, the slug will not be automatically updated — you must update it explicitly.
</Note>

## Request

<ParamField header="Authorization" type="string" required>
  Bearer token for an admin or content-manager account.

  Format: `Bearer <token>`
</ParamField>

<ParamField header="Content-Type" type="string" required>
  Must be `application/json`.
</ParamField>

### Body

<ParamField body="data" type="object" required>
  Wrapper object required by the Strapi v5 REST API.

  <Expandable title="properties" defaultOpen>
    <ParamField body="data.title" type="string" required>
      Display name of the category shown in the menu.

      Example: `"Cappuccino"`
    </ParamField>

    <ParamField body="data.order" type="number" required>
      Integer that determines the display position of this category relative to others. Lower values appear first.

      Example: `5`
    </ParamField>

    <ParamField body="data.icon" type="number" required>
      The numeric ID of an image file already uploaded to the Strapi Media Library. Only images are accepted.
    </ParamField>
  </Expandable>
</ParamField>

<RequestExample>

```bash cURL
curl --request POST \
  --url 'https://api.cafeacacias.com/api/categories' \
  --header 'Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...' \
  --header 'Content-Type: application/json' \
  --data '{
    "data": {
      "title": "Cappuccino",
      "order": 5,
      "icon": 21
    }
  }'
```

```javascript JavaScript
const response = await fetch('https://api.cafeacacias.com/api/categories', {
  method: 'POST',
  headers: {
    Authorization: 'Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    data: {
      title: 'Cappuccino',
      order: 5,
      icon: 21
    }
  })
});

const { data } = await response.json();
```

</RequestExample>

## Response

<ResponseField name="data" type="object">
  The newly created category object.

  <Expandable title="Category object" defaultOpen>
    <ResponseField name="id" type="number">
      Unique numeric identifier assigned to the new category.
    </ResponseField>

    <ResponseField name="documentId" type="string">
      Strapi v5 document identifier.
    </ResponseField>

    <ResponseField name="title" type="string">
      Display name as provided in the request.
    </ResponseField>

    <ResponseField name="slug" type="string">
      URL-friendly identifier auto-generated from `title`.
    </ResponseField>

    <ResponseField name="order" type="number">
      Display position as provided in the request.
    </ResponseField>

    <ResponseField name="createdAt" type="string">
      ISO 8601 timestamp of creation.
    </ResponseField>

    <ResponseField name="updatedAt" type="string">
      ISO 8601 timestamp of last update.
    </ResponseField>

    <ResponseField name="publishedAt" type="string">
      ISO 8601 timestamp of publication. Set automatically on creation.
    </ResponseField>
  </Expandable>
</ResponseField>
